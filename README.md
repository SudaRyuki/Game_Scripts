## 注目してほしいソースコード
本プロジェクトにおいて、私が特にこだわって実装した処理は以下の通りです。

### 1.Photon PUN 2を用いたオンライン同期とホスト権威
* **該当ファイル:** PlayerScript/Player.cs, ItemScript/PlayerItemCollector.cs (またはアイテム取得処理部分)
* **工夫した点:** ネットワーク遅延による「当たった・当たっていない」の矛盾を防ぐため、当たり判定やアイテムの取得処理・破棄を各クライアントではなくホストに集約する「ホスト権威」の設計にしました。また、通信時のカクつきを無くすため、IPunObservable を用いてキャラクターの向きやシールド状態を手動で同期し、滑らかな挙動を実現しています。AIを活用してベースを構築しつつ、通信タイミングのズレによるバグは、自らログを解析してロジックを修正しました。
  
### 2.コルーチンを用いた複雑な状態遷移（UFO連れ去りギミック）
* **該当ファイル:**PlayerScript/Player.cs (AbductionCoroutine, ReturnCoroutine)
* **工夫した点:** UFOに連れ去られ、一定時間後にエイリアンパワーを獲得して帰還するという、対戦の幅を変えるギミックを実装しました。オンライン環境では、帰還時の座標移動が自動補間されて「高速でスライド移動してしまう」というバグが発生しましたが、移動の瞬間だけ一時的に PhotonRigidbody2DView を無効化することで解決しました。また、攻撃モーション中に連れ去られた際のアニメーションの固まりを防ぐため、帰還時に animator.Rebind() を用いて状態を強制リセットする処理を組み込んでいます。

### 3.BaaS (LootLocker) を利用した非同期のランキングシステム
* **該当ファイル:**Script/LootLockerManager.cs
* **工夫した点:**対戦のモチベーションを高めるため、LootLocker APIを導入して勝利数の世界ランキング機能を実装しました。デバイス固有IDを用いた自動ログインやユーザー名の登録処理を非同期で行い、ランキング取得時には、取得したリストとローカルの保存データを照合し、自身の順位をテキストでハイライト表示するUIロジックを構築しました。
